3.3 First Model

From the univariate analysis we pick those with p < 0.2 and excluded: Race, Decreased.Libido, Snore, Daytime.Sleepiness; once these have exagerated OR values.

Remaining variables are: AF, Age.Cat, Alcohol, CFA, CHF, Conc.decrease, Diabetes, Driver, Dyslipidemia, ESS.Cat, GE.Reflux, Gender, Genetics, Humor, Hypertension, Hypothyroidism, MI, NC.Cat, Nocturia, PHT, Renal.Failure, Stroke, VehicleCrashes and Wit.Apneas

For logistic regression models we use a new data set only with complete cases: 411 lines and 25 colunms.
